# Samay Documentation

Documentation for **Samay** — a productivity tracking tool that tracks your activity and shows how much time you spend on applications.

- **Web app:** [getsamay.vercel.app](https://getsamay.vercel.app)  
- **Insights:** [getsamay.vercel.app/insights](https://getsamay.vercel.app/insights)  
- **Frontend repo:** [github.com/chenchunaidu-proximity/samay-fe](https://github.com/chenchunaidu-proximity/samay-fe)

## Contents

- **Getting started** — Introduction, Quickstart, Setup guide, Connect to desktop  
- **Features** — Dashboard, Tasks, Insights, Projects, Account, Admin Dashboard  
- **How-to guides** — View time by app, Manage projects, Use Insights  

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint):

```bash
npm i -g mint
```

From the repo root (where `docs.json` is):

```bash
mint dev
```

Preview at `http://localhost:3000`.

## Publishing

Use the Mintlify GitHub app from your [dashboard](https://dashboard.mintlify.com/settings/organization/github-app) to deploy. Pushes to the default branch deploy automatically.
